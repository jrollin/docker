# About

List of useful scripts to launch docker containers


## Usage

## Node

Npm install

```bash
./node.sh npm install
```

Run index.js in current directory

```bash
./node.sh node index.js
```

## Postgres database

Start postgres server with postgres:password on port 5432

```bash
./db_postgres.sh
```

## Golang

run go file

```bash
./go.sh run hello.go
```
